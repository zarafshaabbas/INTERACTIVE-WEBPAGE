# 🌐 Interactive Webpage (Inline CSS)

## 📌 Project Overview

This project is a simple interactive webpage created using **HTML with inline CSS**. It demonstrates how styling can be applied directly within HTML elements to design a visually appealing page.

---

## ✨ Features

* Attractive background color
* Styled heading and paragraphs
* Responsive image with border and rounded corners
* Clickable button (styled link)
* Opens external profile in a new tab

---

## 🛠️ Technologies Used

* **HTML5** – for webpage structure
* **Inline CSS** – for styling directly inside HTML tags

---

## 📂 Project Structure

```
project-folder/
│
└── index.html   # Main HTML file
```

---

## 🚀 How to Run the Project

1. Save the file as `index.html`
2. Open it in any web browser (Chrome, Edge, Firefox)
3. View the styled webpage

---

## 📸 What the Page Contains

* A welcome heading
* A paragraph with customized text style
* An image with border and rounded edges
* A clickable button linking to a profile

---

## 🎯 Learning Outcomes

* Understanding basic HTML structure
* Applying styles using inline CSS
* Using tags like `<h1>`, `<p>`, `<img>`, `<a>`
* Creating button-like links using `<a>` tag
* Improving page appearance using colors and spacing

---

## 🔮 Future Improvements

* Replace inline CSS with internal or external CSS
* Add JavaScript for interactivity
* Improve responsiveness for mobile devices
* Add more sections like About Me, Projects, and Contact
